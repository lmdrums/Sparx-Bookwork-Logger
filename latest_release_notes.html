Sparx Bookwork Logger v1.1.0

The MSI update!

Tired of bookwork checks?

This project was designed to minimise the effort you have to put in to your Sparx homework.

Getting simple questions and having to write them down question after question can get annoying. This program automatically takes a screenshot after every question and saves it to a folder.

Whenever a bookwork check comes up, it retrieves that image and displays it in the GUI.

Release Notes:

Release Notes for Sparx Bookwork Logger v1.1.0

  • Created a .msi Windows installer for installation of the application to make it easier.
  • Updated README with .msi section instead of .exe and tweaked a few bits (added install Git to Python section of installation).
  • Updated code - fixed a bug where bookwork checks didn't come up automatically in the GUI.
  • Updated notification engine to win11toast as some issues were run into when using notify-py from Win11.
  • Changed .env and .txt file to single .ini settings file for ease.
  • Added feature when you type in a bookwork code into the search box. If the length of the bookwork code is greater than 2, it will highlight the border red.
  • Got rid of the need to reload window after editing settings.
  • Fixed some bugs with resource_path.
